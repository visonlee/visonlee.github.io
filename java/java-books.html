    # java阅读书单

    按照自己的想法,列出一系列的阅读书单,建议按照顺序一步一步来,达到循序渐进的相关,看完一本书之后,提出问题,善于思考总结。

    • MyBatis技术内幕 (opens new window) 🤔 mybatis架构和执行流程, mybatis配置类,mybatis缓存, mybatis内置的数据源原理, mybatis插件等等

    • spring揭秘 (opens new window) 🤔 spring有哪些模块,架构,执行流程，AOP,IOC原理 mybatis放到spring中去管理,spring是怎样去管理mybatis的

    • 架构探险——从零开始写Java Web框架 (opens new window) 通过这么书,实现一个简单的web框架,了解其中的原来,这样更能容易理解spring mvc中的代码

    • 看透Spring MVC:源代码分析与实践 (opens new window) 还没看,希望能够通过这本书了解spring mvc的原理和执行流程

    • 精通Spring 4.x (opens new window) 很详细

    • spring boot编程思想 (opens new window) 要知道如何从spring，spring mvc过渡到spring boot,了解自动配置原理等等

    • 深入剖析Tomcat (opens new window) 了解tomcat内部的工作原理
